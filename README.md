# samplepoetry

Test poetry repo 

## Installation

Install with pip (not checked):

`pip install samplepoetry`

## Testing

`poetry run tox` <br/>
`poetry run flake8`

or 

`poetry shell` to activate shell (also `. venv.sh`) then <br/>
`tox` or `flake8` inside of venv


## Running

Run the monitor with:

`poetry run samplepoetry`
